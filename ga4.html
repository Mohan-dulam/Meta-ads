Google Analytics 4 – Data Measurement System

flowchart TD
UserAction --> GA4Tag
GA4Tag --> Event
Event --> GA4Server
GA4Server --> Reports
GA4Server --> Attribution

Key Concepts

  • Event: Any interaction
  • Parameter: Extra event detail
  • Conversion: Marked important event

Default Ecommerce Events

  • view_item
  • add_to_cart
  • begin_checkout
  • purchase

Attribution

GA4 uses data-driven attribution (not last click).

Example

User sees Facebook ad → searches on Google → purchases

GA4 credits both channels.
