Meta Pixel & Conversion Tracking

What is Meta Pixel?

A JavaScript tracking code placed on the website to record user actions.

Standard Events

  • PageView
  • ViewContent
  • AddToCart
  • InitiateCheckout
  • Purchase
  • Lead

Why Pixel is Important

  • Conversion optimization
  • Retargeting
  • Lookalike audience creation

Aggregated Event Measurement

Due to iOS14 privacy updates Meta tracks only prioritized events.

Conversion API

Server-side tracking that sends events directly from server to Meta, improving tracking accuracy.
