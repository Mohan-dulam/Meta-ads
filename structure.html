Meta Ads Delivery System (How Ads Actually Appear to a User)

Meta does NOT show ads based only on budget. It runs an auction system every single time a user opens Facebook or Instagram.

Ad Delivery Flow

flowchart TD
U[User Opens Instagram/Facebook] --> R[Meta Checks Available Ads]
R --> A[Eligible Ads Selected]
A --> B[Meta Auction Runs]
B --> C[Ad Rank Calculated]
C --> D[Highest Value Ad Wins]
D --> E[Ad Appears to User]

What Determines the Winner? (Ad Rank)

Meta calculates a score called Total Value.

Total Value = Bid + Estimated Action Rate + Ad Quality
  • Bid: What advertiser is willing to pay
  • Estimated Action Rate: Probability user will click/convert
  • Ad Quality: Feedback, engagement, hide rate
Important:

Highest budget does NOT always win. Best predicted performance wins.

Backend Example

Two advertisers target same user:

  • Advertiser A: ₹200 bid, poor creative
  • Advertiser B: ₹120 bid, highly engaging creative
Meta predictse: Estimated Action Rate higher for B → B wins auction.

Campaign → Ad Set → Ad Responsibilities

  • Campaign: Defines optimization goal (leads, purchases)
  • Ad Set: Audience, budget, schedule, placements
  • Ad: Creative & messaging

Learning Phase

Meta tests your ad across multiple users to find who converts best. Needs ~50 conversions/week per ad set.
